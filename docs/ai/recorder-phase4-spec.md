# インタラクティブレコーダー Phase 4 仕様

ADR-009 Phase 4: 録画の編集・再生・微調整

## 概要

Phase 2-3 で録画・アサート機能が実装されたが、録画後のステップ編集体験が不十分。
Phase 4 では、録画したステップの **並び替え・複製・中間挿入** を追加し、実用的なテスト作成ワークフローを完成させる。

## 機能一覧

### 1. ステップのドラッグ&ドロップ並び替え

DetailPanel のステップリストで、ステップをドラッグして順序を変更できる。

**操作:**
- ステップ左端のドラッグハンドル（⠿）を掴んでドラッグ
- ドロップ先にインジケーター線を表示
- ドロップ時に `order` を自動再採番

**技術方式:**
- HTML5 Drag and Drop API を使用（外部ライブラリ不要）
- `draggable` 属性 + `onDragStart/onDragOver/onDrop` イベント
- ドラッグ中の要素は半透明（`opacity: 0.4`）
- ドロップ先は上側に 2px のアクセント色ラインを表示

### 2. ステップの複製

既存ステップをワンクリックで複製し、直後に挿入する。

**操作:**
- ステップ上の「⧉」ボタンをクリック
- 複製されたステップが直後に挿入される
- `order` は自動再採番

### 3. ステップの中間挿入

ステップとステップの間に新しいステップを挿入できる。

**操作:**
- ステップ間に「+ 挿入」ボタンをホバーで表示
- クリックすると空のステップが挿入され、編集モードに入る

## UI 変更

### DetailPanel ステップリスト

```
┌─────────────────────────────┐
│ ⠿ 1  navigate  /login      │  ← ドラッグハンドル + 複製ボタン
│                         ⧉  │
├─ ─ ─ + 挿入 ─ ─ ─ ─ ─ ─ ─ ┤  ← ホバーで表示
│ ⠿ 2  type  #email → ...    │
│                         ⧉  │
├─ ─ ─ + 挿入 ─ ─ ─ ─ ─ ─ ─ ┤
│ ⠿ 3  click  button.submit  │
│                         ⧉  │
└─────────────────────────────┘
[+ ステップ追加]                 ← 既存（末尾追加）
```

### ドラッグ中の表示

- ドラッグ元: `opacity: 0.4`、背景色維持
- ドロップ先: ステップの上辺に `2px solid var(--color-accent)` のライン
